## 核心流程

### 1. 文档入库流程

```
POST /api/v1/kb/documents
        │
        ▼
  ┌──────────────┐     ┌──────────────┐     ┌──────────────┐
  │ 文件接收与校验 │ ──→ │ 异步任务队列  │ ──→ │ 文档解析      │
  │ (大小/类型)   │     │ (asyncio)    │     │ txt/md/pdf/docx│
  └──────────────┘     └──────────────┘     └──────┬───────┘
                                                    │
                    ┌───────────────────────────────┘
                    ▼
  ┌──────────────┐     ┌──────────────┐     ┌──────────────┐
  │ 文本清洗      │ ──→ │ 智能切片      │ ──→ │ Embedding     │
  │ 去噪/去重    │     │ 固定+语义     │     │ BGE 1024维    │
  └──────────────┘     └──────────────┘     └──────┬───────┘
                                                    │
                    ┌───────────────────────────────┘
                    ▼
  ┌──────────────┐
  │ pgvector 写入 │  (chunks + embeddings + HNSW 索引构建)
  │ PostgreSQL    │
  └──────────────┘
```

### 2. 智能问答流程

```
POST /api/v1/kb/chat
        │
        ▼
  ┌──────────────┐   SSE progress: intent
  │ 1. 意图识别   │   最简单的 keyword/长度推断（P0），不做复杂 NLU
  └──────┬───────┘
         ▼
  ┌──────────────┐   SSE progress: retrieval
  │ 2. Query向量化│   BGE Embedding → 1024维向量
  └──────┬───────┘
         ▼
  ┌──────────────┐   SSE progress: matching
  │ 3. HNSW检索   │   pgvector cosine_distance + Top-K + threshold 过滤
  └──────┬───────┘
         ▼
  ┌──────────────┐
  │ 4. 无匹配？   │   YES → SSE error: "未找到相关知识"
  │    similarity │   NO  → 继续
  │    < threshold│
  └──────┬───────┘
         ▼
  ┌──────────────┐
  │ 5. Prompt组装 │   系统提示词 + 知识片段 + 对话历史 + 用户问题
  └──────┬───────┘
         ▼
  ┌──────────────┐   SSE progress: generating
  │ 6. LLM 调用   │   httpx → Ollama API (stream=True)
  │    SSE 流式   │   ──→ SSE token × N
  └──────┬───────┘
         ▼
  ┌──────────────┐   SSE done: { ref_chunks }
  │ 7. 结果持久化 │   kb_qa_record INSERT
  └──────────────┘
```

### 3. 检索测试流程

```
POST /api/v1/kb/retrieval/test
        │
        ▼
  ┌──────────────┐
  │ Query 向量化  │
  └──────┬───────┘
         ▼
  ┌──────────────┐
  │ HNSW 检索     │  (可使用自定义 top_k / threshold，覆盖 config 默认值)
  └──────┬───────┘
         ▼
  ┌──────────────┐
  │ 返回结果      │  JSON: { results[], elapsed_ms }
  └──────────────┘
```

## 数据库连接管理

```python
# app/core/database.py

from sqlalchemy.ext.asyncio import create_async_engine, async_sessionmaker
from app.config import settings

engine = create_async_engine(
    settings.DATABASE_URL,
    echo=settings.LOG_LEVEL == "DEBUG",
    pool_size=10,
    max_overflow=20,
)

AsyncSessionLocal = async_sessionmaker(engine, expire_on_commit=False)

async def get_db():
    async with AsyncSessionLocal() as session:
        yield session
```
